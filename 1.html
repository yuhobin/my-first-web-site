Hypertext Markup languges (HTML) is the standard markup languges for creating web pages and applications. Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the struture of a web page semantically and originally included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. with HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page. It provides a means to create structred documents by denoting strutual sematics for text such as headings, paragraphs, lists, links, quotes and other items. HTML elemrnts are delineated by tags, written using angle brackets.
